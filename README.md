# 日日 (Nichinichi) - 日語學習 App

一款以**日式禪意美學**為核心的日語假名學習應用程式，專為繁體中文使用者打造。

## 專案簡介

「日日」致力於提供沉浸式的日語假名學習體驗，結合極簡設計與實用功能，讓學習過程如同禪修般寧靜而專注。

## 核心功能

### 📚 假名學習系統
- **完整五十音圖表**
  - 清音 (Seion)：あ行～わ行
  - 濁音/半濁音 (Dakuon/Handakuon)：が行～ぱ行
  - 拗音 (Youon)：きゃ、しゃ、ちゃ 等組合
- **平假名/片假名切換**：一鍵切換學習模式
- **羅馬拼音標註**：使用 Ruby Text 呈現讀音
- **語音朗讀 (TTS)**：點擊即可聆聽標準發音

### ✍️ 書寫練習 (Zen Canvas)
- **禪意筆觸畫布**：模擬毛筆書寫質感
- **多字元網格系統**：支援拗音等多字元練習
- **即時清除功能**：快速重新練習

### 📝 測驗系統
- **自訂測驗範圍**：選擇特定假名類型進行測驗
- **即時回饋**：答題後立即顯示正確答案
- **熟練度追蹤**：記錄學習進度與答題統計
- **測驗結果分析**：詳細的答題報告與建議

### 🎨 Zen UI 設計系統
- **極簡主義介面**：去除視覺雜訊，專注學習本質
- **深色模式支援**：跟隨系統自動切換
- **和諧配色**：使用 HSL 色彩系統打造舒適視覺
- **流暢動畫**：微互動提升使用體驗
- **自訂 Zen 元件**：ZenButton、ZenCard、ZenToast 等
